在你的 App 中使用《一拳超人》背景色

发表于 2016-03-14

这是 Github 项目地址


我从《一拳超人》动画中吸取了22种和白色非常匹配的颜色，现在你可以通过 UIColor 或者 UIView 来通过一行代码调用这些颜色。

// quick version
self.view.punch()

// slow version
self.view.punchWithInterval(2)

// complete version
self.view.backgroundColor = UIColor.onePunchColor()
